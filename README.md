MeasuringUx
======
[![Build Status](https://travis-ci.org/crispab/MeasuringUx.svg?branch=master)](https://travis-ci.org/crispab/MeasuringUx)

MeasuringUx visualizes Ux metrics. 

It's built with
- Play Framework 2.4.x
- Java 8
- Chart.js 1.0.2
- Bootstrap 3.3.5
